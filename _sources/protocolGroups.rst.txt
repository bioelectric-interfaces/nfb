Protocol groups
=====================

.. image:: groups.png
   :width: 600
   
This module allows you to organize a group of protocols.

**Name**: name of the group.

**Protocols**: a list of previously added protocols that will be included to the group.

**Corresponding quantity**: a list of the number of repetitions of the corresponding protocols.

**Order**: "Shuffle" enables / disables randomness in the sequence.

**Split by protocol**: name of the protocol that will split sequence.
